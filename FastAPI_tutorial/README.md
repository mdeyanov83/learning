## Environment Setup

### Install UV (package + project manager):
* pip install uv

### Install dependencies:
* uv add package_name

    - fastapi
    - python-dotenv -> package to manage and load environment variables
    - fastapi-users[sqlalchemy] -> package
    - imagekitio
    - uvicorn[standard] -> Web server
